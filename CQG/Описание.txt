Файл index.html состоит из 3 элементов:
1. Canvas для отображения фигур;
2. TextArea для ввода описания фигуры в текстовом формате;
3. Кнопка "Draw" для отрисовки введенных фигур.

Формат описания фигуры:

Figurename -p [i, j]... options
где figureName - имя фигуры,
–p [x, y]... - координаты основных точек фигуры, options - дополнительные параметры и опции фигуры.

Имена поддерживаемых фигур:

1. line 
2. rectangle
3. triangle
4. circle
5. ellipse

Параметры: 

-p : координаты фигуры
-r : радиус для окружности
-r1 и -r2 : радиусы r1 и r2 для эллипса
-с : цвет контура фигуры в формате rgb
-b : цвет заливки фигуры в формате rgba

Обязательные параметры:

обязательные параметры для фигур указываются внутри квадратных скобок [ ]
Пример:
-p [200, 300] -r [25]

1. line : обязательные параметры -p [nX, nY] [mX, mY],
	где nX и nY - начальные координаты линии;
	mX и mY - конечные координаты линии. 

2. rectangle: обязательные параметры: -p [nX, nY] [mX, mY],
	где nX и nY - начальные координаты квадрата;
	mX и mY - конечные координаты квадрата.

3. triangle: обязательные параметры: -p [nX, nY] [mX, mY] [kX, kY],
	где nX и nY - координаты первыой вершины треугольника;
	mX и mY - координаты второй вершины треугольника;
	kX, kY - координаты третьей вершины треугольника;

4. circle : обязательные параметры -p [nX, nY] -r [m],
	где nX и nY - координаты центра окружности;
	m - радиус окружности
5. ellipse : обязательные параметры -p [nX, nY] -r1 [m] -r2 [k],
	где nX и nY - координаты центра эллипса;
	m и k - радиус эллипса r1 и r2 соответственно.

Опциональные параметры:

Для каждой фигуры может быть указан один или несколько опциональных параметров, которые указываются в круглых скобках ().

Пример:
1. -c (255, 0, 255);
2. -b (200, 0, 0, 0.3)

Фигура line поддерживает только один дополнительный параметр (-с),
остальные фигуры поддерживают оба дополнительных параметров (-с, -b).

Если дополнительный параметр не указан явно, то будут использованы значения по умолчанию:

-c (0, 0, 0);
-b (0, 0, 0, 0).
